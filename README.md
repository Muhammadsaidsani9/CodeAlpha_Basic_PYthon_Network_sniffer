# Python Packet Sniffer
Simple network packet sniffer using Scapy and Tkinter.  
Supports GUI and CLI modes for live traffic capture.

## Features
- Live packet capture with source/dest IP, protocol, ports
- GUI mode with real-time table updates
- CLI mode for lightweight sniffing
- Thread-safe, responsive GUI

## Setup
1. Install Npcap on Windows with WinPcap API mode enabled
2. Run: `pip install -r requirements.txt`
3. Run as Administrator: `python main.py`
4. Choose `2` for GUI or `1` for CLI
